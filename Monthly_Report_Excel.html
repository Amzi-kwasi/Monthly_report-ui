Terms of use
Welcome to this platform. This platform is a user-friendly platform.It allows you to input numbers and sum them for you. Just key in, and the input will be saved.
How it saves data

This platform is set to save your input to localstorage. These means when you clear your Browser's cache, it will affect the data.

Restrictions

You may not use any part of the web page, software, or its data to write another website or software product, whether this is done in exchange for money or not.

How to use

The input with the placeholder "HRS","VID","R.V","PLC" and "B.S" can be change to your own text.

The month uses your system date and time changes so make sure your date and time are set properly

The input created only allows you to input 'numbers'. So just input the number.

Your input is saved automatically, so dont stress yourself saving anything.

To see all the input field click on the "Max" button.

Your input sum up automatically to the "Total_Report" respectively.

You can also "Add Note" if needed.

In the "Total_Report" container, you can change the names in the input to your own choice. NB:Make sure the name in the Total_Report matches with the table's top names..

By clicking the "clear all data" button , you agree to delete your data permanently from localstorage

You can use clear all data button when you enter a new month

Enjoy your day.
